# Development Workspace

A flexible development workspace setup that supports multiple programming languages and frameworks.

## Project Structure

```
sterl/
├── .github/
│   └── copilot-instructions.md    # Copilot workspace instructions
├── src/                           # Source code
│   ├── hello.py                   # Python Hello World example
│   └── hello.js                   # JavaScript Hello World example
├── docs/                          # Documentation
├── tests/                         # Test files
└── README.md                      # This file
```

## Getting Started

This workspace includes sample "Hello World" programs in multiple languages:

### Python
```bash
python3 src/hello.py
```

### JavaScript (Node.js)
```bash
node src/hello.js
```

## Development

This workspace is set up to support various types of development projects. You can:

1. Add new source files in the `src/` directory
2. Add documentation in the `docs/` directory  
3. Add tests in the `tests/` directory
4. Configure build tools and dependencies as needed
